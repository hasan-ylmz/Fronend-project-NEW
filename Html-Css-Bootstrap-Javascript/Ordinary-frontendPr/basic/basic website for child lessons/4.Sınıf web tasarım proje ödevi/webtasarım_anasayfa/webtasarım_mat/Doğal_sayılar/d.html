Doğal Sayılar

Doğal Sayılar

Sıfırdan başlayıp birer birer artarak sonsuza kadar giden sayılara doğal sayılar denir. Doğal sayılar kümesi “N” ile gösterilir. N = {0, 1, 2, 3, …}

Sayma Sayılar

Birden başlayıp birer birer artarak sonsuza kadar giden sayılara sayma sayılar denir. Sayma sayılar kümesi “S” ile gösterilir. S = {1, 2, 3, …}

Doğal Sayılar Kümesinde Toplama İşleminin Özellikleri
1. Değişme Özelliği

Doğal sayılar kümesinde toplama işleminin değişme özelliği vardır. Doğal sayılarda toplama işlemi yaparken toplanan sayıların yerleri değiştiğinde toplama işleminin sonucu değişmez.

2. Birleşme Özelliği

Doğal sayılar kümesinde toplama işleminin birleşme özelliği vardır. Üç doğal sayı toplanırken toplananların herhangi ikisi önce toplanırsa sonuç değişmez.

3. Etkisiz Eleman

Toplama işleminin etkisiz elemanı sıfırdır. Bir doğal sayının “0” ile toplamı sayının kendisine eşittir. Örnekler 7 + 0 = 7 0 + 7 = 7’dir.
